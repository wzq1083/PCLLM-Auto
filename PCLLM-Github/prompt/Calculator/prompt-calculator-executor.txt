一个字典存储了软件的按钮附属关系，例如{"1": {"2": "5": {}， "3": "6": {}, "4": {}}}，意思是，当点击按钮1时，展开新画面可以看到按钮2,按钮3和按钮4，在点击按钮2时可以展开看到按钮5，在点击按钮3时可以展开看到按钮6. 点击按钮4，5，6时则不会展开任何新按钮。
记事本的按钮附属字典为：
calculator_dict =
{
    "mode select":
     {
        "standard":
        {
            "1": {},    "2": {},  "3": {},  "4": {},   "5": {},   "6": {},    "7": {},    "8": {},  "dot":{},
            "9": {},    "0": {},    "+":{}, "-": {},    "multi": {},    "division": {}, "=": {},    "delate":{},
            "clear":{}  ,"clear entry":{},  "reciprocal":{},    "square":{},   "sqrt":{}, "MC":{}, "MR":{}, M+":{}, "M-":{}, " Ms":{},
            "memory":{}, "history":{}
        }
 

在计算器里，这些按钮的作用是
mode select:
    standard: 计算器切换到标准模式
        1: 输入数字1。
        2: 输入数字2。
        3: 输入数字3。
        4: 输入数字4。
        5: 输入数字5。
        6:  输入数字6。
        7: 输入数字7。
        8: 输入数字8。
        9: 输入数字9。
        0: 输入数字0。
        +: 输入运算符加号。
        -: 输入运算符减号。
        multi: 输入运算符乘号。
        division: 输入运算符除号
        =: 输入等号
        dot: 输入小数点
        delate: 删除刚刚输入的一个数
        clear: 清除计算器的所有输入和显示。
        clear entry: 被设计用来清除当前输入的数字或操作，而不会影响已计算的结果
        reciprocal: 倒数键，先输入分母，然后再按下此键即可得到该分母的倒数
        square: 平方键，先输入一个数，然后再按下此键即可得到该数字的平方
        sqrt: 开方键，先输入一个数，然后再按下此键即可得到该数字的开平方后的结果
	MC : 清除存储器中的所有内容。
	MR : 调用存储器中存储的数值并显示出来。
	M+ : 将当前显示的数值加到存储器中已存储的数值上。
	M- : 将当前显示的数值从存储器中已存储的数值中减去。
	MS : 将当前显示的数值保存到存储器中。


快捷键设置：
	Alt + 1: 切换到标准模式（Standard）。
	Alt + 2: 切换到科学模式（Scientific）。
	Alt + 3: 切换到图形模式（Graphing）。
	Alt + 4: 切换到编程模式（Programmer）。
	Alt + 5: 切换到日期计算模式（Date Calculation）。

我之后提到的input()只是click()操作的集合，只是为了方便表示，例如：input(45)意思是指click(4)->click(5),你在输出的时候注意，不要直接输出input(45)，你要先转换为click操作再计算
1. **模式切换**：点击计算器的菜单项切换到所需模式，包括基本、科学、程序员、日期计算等模式。
   - 格式:click(mode select)->click(Scientific)表示切换到科学计算模式。

2. **基本计算**：执行加、减、乘、除等运算。
   - 格式:input(25+30)->click(=)表示输入表达式并按回车计算。

3. **存储器使用**：保存中间运算的结果到存储器，需要时调用存储器中的结果
  -使用案例 例如计算150-10*2时，需要先将10*2的结果存储，后续再调用
  -对应操作 input(10*2)->click(MS)->input(150)

8. **单目运算**：
   例如指令让你求5的单目运算
   - 开方运算:click(5)->click(sqrt)
   - 倒数运算:click(5)->click(reciprocal)
   - 平方运算:click(5)->click(square)


注意事项：
- **快捷键操作**：如果有快捷键可用，必须优先使用快捷键。例如，切换模式可能有快捷键（如Alt+1切换到基本模式）。
- **输入格式**：输入内容时，严格按照计算器支持的格式。例如：
  - 输入表达式:input(25+30)。
  - 输入单位转换:input(10km)。
  - 输入日期:input(2024-11-19)。
-**重置输入**：
   - 在每次执行完一次模式切换后都要进行一次清零操作，即点击一次清屏键
   - 但是有一个例外，即在科学模式下时，要先点一次clear，再点一次clear


输出格式化操作，例如：
click(mode select)->click(Scientific)->click(sin)->click(4)->click(=)
