一个字典存储了软件的按钮附属关系，例如{"1(Ctrl+J)": {"2": "5": {}， "3(Ctrl+C)": "6": {}, "4": {}}}，意思是，当点击按钮1时，展开新画面可以看到按钮2,按钮3和按钮4，在点击按钮2时可以展开看到按钮5，在点击按钮3时可以展开看到按钮6. 点击按钮4，5，6时则不会展开任何新按钮。若一个按钮后有括号，则括号中的内容为对应功能的快捷键，如按组合键Ctrl+J相当于点击按钮1

写字板中，这些按钮的作用是
   "document": 文件相关设置
	"new_tab":新建一个tab,
	"save":保存文件内容,
	"save as":另存文件
		"rtf":另存为rtf格式,
		"xml":另存为xml格式, 
		"txt":另存为txt格式,
	"exit":退出wordpad
    
  "homepage": 主页
	"paste":粘贴剪切板中内容,
	"font":调节字体,
	"font size":调节字号,
	"enlarge font":增大字号,
	"reduce font size":减小字号,
	"overbold" : 加粗,
	"tilt" : 斜体,
	"underline" : 下划线,
	"strikeout line" : 删除线,
	"subscript" : 添加下标,
	"superscript" : 添加上标,
	"text color" : 设置文本颜色
		"text-red":红色 
		"text-blue":蓝色, 
		"text-grey":灰色,
		"text-green":绿色, 
		"text-yellow":黄色, 
		"text-orange":橘色
	"text background color" : 设置文字背景颜色
		"background-red":红色 
		"background-blue":蓝色, 
		"background-grey":灰色,
		"background-green":绿色, 
		"background-yellow":黄色, 
		"background-pink":粉色
	"reduce retract":减小缩进,
	"add retract":增大缩进,
	"left-aligned":左对齐,
	"right-aligned":右对齐,
	"middle-aligned":中间对齐,
	"justified-aligned"两端对齐,
    
    "view": 
	"zoom in":放大视图,
	"zoom out": 缩小视图,
	"default zoom":默认视图,
	"ruler":标尺,
	"status bar":状态栏,
	"auto line break":自动换行
		"no auto line break":不自动换行， 
		"by window":按窗口自动换行, 
		"by ruler":按标尺自动换行,
	"unit":单位
		"inch":英尺， 
		"centimeter":厘米, 
		"pound":磅

你可以进行的操作包含：
	点击按钮：格式化输出为click(1)代表点击按钮1
	双击操作：格式化输出为doubleclick(1)代表双击按钮1
	按键操作：格式化输出为press(right)代表按下右键
	输入文本：格式化输出为input(text)代表输入text文本

特殊地，一些复杂操作的示例如下
    另存文件：如将文件另存为"new-file.txt"
	click(document)->click(save as)->click(txt)->input(new-file)->press(enter)
	注意操作第三步要根据要求的文件格式合理选择按钮
    
    更改字体：如将字体设置为"wide latin"
	click(homepage)->click(font)->input(wide latin)->press(enter)
	注意第三步要根据要求的字体输入响应的文本

    更改字号：如将字号设置为"30"
	click(homepage)->click(font size)->input(30)->press(enter)
	注意第三步要根据要求的字号输入响应的文本

    更改字体颜色：如将字体颜色更改为黄色
	click(homepage)->click(text color)->click(text-yellow)
	注意第三步要根要求点击响应的颜色按钮
    
    增大字号：如将字号增大
	click(homepage)->click(enlarge font)

    更改字体背景颜色：如将字体颜色更改为黄色
	click(homepage)->click(text background color)->click(background-yellow)
	注意第三步要根要求点击响应的颜色按钮

    在wordpad输入内容：如输入“Hello World!”
	click([1000, 500])->input(Hello World!)
	首先点击（1000， 500）像素位置是为了将光标移动到wordpad写字板里

上述内容仅仅是为了让你能够明白写字板的按钮架构以及简单用法，不需有任何多余输出
