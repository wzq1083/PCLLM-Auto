一个字典存储了软件的按钮附属关系，例如{"1": {"2": "5": {}， "3": "6": {}, "4": {}}}，意思是，当点击按钮1时，展开新画面可以看到按钮2,按钮3和按钮4，在点击按钮2时可以展开看到按钮5，在点击按钮3时可以展开看到按钮6. 点击按钮4，5，6时则不会展开任何新按钮。

特殊地，某些按钮具有特定的快捷键。具体地，Cut的快捷键ctrl+x，Copy的快捷键ctrl+c，Paste的快捷键ctrl+v，Find的快捷键ctrl+f，Replace的快捷键ctrl+h，New tab的快捷键ctrl+n，New Window的快捷键ctrl+shift+n，Open的快捷键ctrl+o，Save的快捷键ctrl+s，Save as的快捷键ctrl+shift+s的快捷键,Save all的快捷键ctrl+alt+s，Print的快捷键ctrl+p，Close tab的快捷键ctrl+w， Close window的快捷键ctrl+shift+w, Undo的快捷键ctrl+z，Find next的快捷键f3，Find previous的快捷键shift+f3，Goto的快捷键ctrl+g，Select all的快捷键ctrl+a，TimeDate的快捷键f5，Zoom in的快捷键ctrl+add，Zoom out的快捷键ctrl+-，Restore default zoom的快捷键ctrl+0，输入对应的快捷键操作相当于直接点击相应的按钮。

记事本中，这些按钮的作用是
File:
    New tab: 打开新的标签页。
    New window: 打开新的窗口。
    Open: 打开文件。
    Save: 保存当前文件。
    Save as: 另存为新文件。
		Save path：输入文件保存的路径。
                Save file name：输入文件命名。
                Save file confirm：确认保存文件。
			Overwrite confirm：确认覆盖重名文件
			Overwrite cancel：取消覆盖重名文件
                Save file cancel：取消保存文件。
    Save all: 保存所有文件。
    Page setup: 页面设置。
    Print: 打印当前文件。
    Close tab: 关闭当前标签页。
    Close window: 关闭当前窗口。
    Exit: 退出记事本应用。
Edit:
    Undo: 撤销上一次操作。
    Cut: 剪切选中的内容。
    Copy: 复制选中的内容。
    Paste: 粘贴剪切板中的内容。
    Delete: 删除选中的内容。
    Find: 查找文本。
        Find box: 查找输入框，可再此输入查找的文本。
        Search up: 向上查找。
        Search down: 向下查找。
    Replace: 替换文本。
        Find box: 查找输入框。
        Search up: 向上查找。
        Search down: 向下查找。
        Replace box: 替换输入框，可在此输入替换文本。
        Replace one: 替换一个匹配项。
        Replace all: 替换所有匹配项。
    Goto: 跳转到指定行。
		Goto linenum clear:清除跳转行数输入框中的数字
		Goto linenum input:输入框，用于输入跳转到的行数  
		Goto confirm:确认按键，点击后跳转到对应行
			Line error OK:如果输入的行数在记事本中不存在，则跳出此按钮。
		Goto cancel:取消按键，点击后取消跳转操作

    TimeDate: 插入当前日期和时间。
View:
    Zoom: 缩放视图。
    Zoom in: 放大视图。
    Zoom out: 缩小视图。
    Restore default zoom: 恢复默认缩放。


你可以进行的操作包含：
	点击按钮：格式化输出为click(1)代表点击按钮1
	双击操作：格式化输出为doubleclick(1)代表双击按钮1
	按键操作：格式化输出为press(right)代表按下右键
	输入文本：格式化输出为input(text)代表输入text文本
	组合键：格式化输出为hotkey(ctrl+c)代表组合键操作ctrl+c
	选中段落：格式化输出为select_para(01,02)代表选中第一段的内容。每个数字均为两位，不足两位的前面补零。其中第一个数字代表起始段，第二个数字代表的段落内容不会被选中
