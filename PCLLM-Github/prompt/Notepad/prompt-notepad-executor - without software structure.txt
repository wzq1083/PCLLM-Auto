特殊地，某些按钮具有特定的快捷键。具体地，Cut的快捷键ctrl+x，Copy的快捷键ctrl+c，Paste的快捷键ctrl+v，Find的快捷键ctrl+f，Replace的快捷键ctrl+h，New tab的快捷键ctrl+n，New Window的快捷键ctrl+shift+n，Open的快捷键ctrl+o，Save的快捷键ctrl+s，Save as的快捷键ctrl+shift+s的快捷键,Save all的快捷键ctrl+alt+s，Print的快捷键ctrl+p，Close tab的快捷键ctrl+w， Close window的快捷键ctrl+shift+w, Undo的快捷键ctrl+z，Find next的快捷键f3，Find previous的快捷键shift+f3，Goto的快捷键ctrl+g，Select all的快捷键ctrl+a，TimeDate的快捷键f5，Zoom in的快捷键ctrl+add，Zoom out的快捷键ctrl+-，Restore default zoom的快捷键ctrl+0，输入对应的快捷键操作相当于直接点击相应的按钮。

记事本中，这些按钮的作用是
File:
    New tab: 打开新的标签页。
    New window: 打开新的窗口。
    Open: 打开文件。
    Save: 保存当前文件。
    Save as: 另存为新文件。
		Save path：输入文件保存的路径。
                Save file name：输入文件命名。
                Save file confirm：确认保存文件。
			Overwrite confirm：确认覆盖重名文件
			Overwrite cancel：取消覆盖重名文件
                Save file cancel：取消保存文件。
    Save all: 保存所有文件。
    Page setup: 页面设置。
    Print: 打印当前文件。
    Close tab: 关闭当前标签页。
    Close window: 关闭当前窗口。
    Exit: 退出记事本应用。
Edit:
    Undo: 撤销上一次操作。
    Cut: 剪切选中的内容。
    Copy: 复制选中的内容。
    Paste: 粘贴剪切板中的内容。
    Delete: 删除选中的内容。
    Find: 查找文本。
        Find box: 查找输入框，可再此输入查找的文本。
        Search up: 向上查找。
        Search down: 向下查找。
    Replace: 替换文本。
        Find box: 查找输入框。
        Search up: 向上查找。
        Search down: 向下查找。
        Replace box: 替换输入框，可在此输入替换文本。
        Replace one: 替换一个匹配项。
        Replace all: 替换所有匹配项。
    Goto: 跳转到指定行。
		Goto linenum clear:清除跳转行数输入框中的数字
		Goto linenum input:输入框，用于输入跳转到的行数  
		Goto confirm:确认按键，点击后跳转到对应行
			Line error OK:如果输入的行数在记事本中不存在，则跳出此按钮。
		Goto cancel:取消按键，点击后取消跳转操作

    TimeDate: 插入当前日期和时间。
View:
    Zoom: 缩放视图。
    Zoom in: 放大视图。
    Zoom out: 缩小视图。
    Restore default zoom: 恢复默认缩放。


你可以进行的操作包含：
	点击按钮：格式化输出为click(1)代表点击按钮1
	双击操作：格式化输出为doubleclick(1)代表双击按钮1
	按键操作：格式化输出为press(right)代表按下右键
	输入文本：格式化输出为input(text)代表输入text文本
	组合键：格式化输出为hotkey(ctrl+c)代表组合键操作ctrl+c
	选中段落：格式化输出为select_para(01,02)代表选中第一段的内容。每个数字均为两位，不足两位的前面补零。其中第一个数字代表起始段，第二个数字代表的段落内容不会被选中

特殊地，一些特殊注意的地方包括：
    查找：	例如要查找“111”
        hotkey(ctrl+f)->click(Find box clear)->click(Find box)->input(111)->点击Search up或Search down跳转到指定的搜索位置->click(Exit find)。
	每一步的含义是：首先快捷键打开查找窗口，接下来清楚Find box中内容，点击Find box，输入被替换的内容，接下来清楚Find box中内容，点击Find box，输入被替换的内容，最后退出查找

    替换：例如要将“111”替换为“222”
        hotkey(ctrl+h)->click(Find box clear)->click(Find box)->input(111)->click(Replace box)->click(Replace box clear)->input(222)->click(Replace one或者Replace all)->click(Exit find)。
	每一步的含义是：首先快捷键打开替换窗口，接下来清楚Find box中内容，点击Find box，输入被替换的内容，点击Replace box输入替换的内容，根据用户需求替换一处或替换全部，最后退出替换功能。如果遇到替换任务，一定要确认点击了替换按钮

    跳转到某行：例如像跳转到第5行
       hotkey(ctrl+g)->click(Goto linenum clear)->click(Goto linenum input)->input(5)->click(Goto confirm)->click(Line error OK)->click(Goto cancel)
	注意要将input里面的内容替换成对应的行数
      	
    另存文件：如将文件保存为111.txt
	hotkey(ctrl+shift+s)->click(Save file name)->hotkey(ctrl+a)->input(111.txt)->press(enter)->click(overlap yes)/click(overlap no)
        注意将input里面的内容替换为用户实际指定的文件名，最后一步选择click(overlap yes)/click(overlap no)要根据用户提出的指令，如果存在同名文件是否覆盖做出相应判断。特別注意，一定不要忽略click(Save path)选择保存路径这个步骤。

    注意注意注意！！！单击File、Edit或View会展开二级按钮，要想点击这些按钮（只包含点击File、Edit或View会展开二级按钮），需要使用doubleclick功能。如想要点击Exit按钮，则顺序式，单击File（click(File)），双击Exit(doubleclick(Exit))，注意注意，doubleclick只在想要点击File、Edit或View会展开二级按钮时才会被需要。尤其注意！！！尤其注意這些二级按钮需要双击才可被点中
