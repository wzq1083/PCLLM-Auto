记事本中，这些按钮的作用是
File:
    New tab: 打开新的标签页。
    New window: 打开新的窗口。
    Open: 打开文件。
    Save: 保存当前文件。
    Save as: 另存为新文件。
	Save path：输入文件保存的路径。
        Save file name：输入文件命名。
        Save file confirm：确认保存文件。
	    Overwrite confirm：确认覆盖重名文件
	    Overwrite cancel：取消覆盖重名文件
        Save file cancel：取消保存文件。
    Save all: 保存所有文件。

    Delete: 删除选中的内容。
    Find: 查找文本。
        Find box: 查找输入框，可再此输入查找的文本。
        Search up: 向上查找。
        Search down: 向下查找。
    Replace: 替换文本。
        Find box: 查找输入框。
        Search up: 向上查找。
        Search down: 向下查找。
        Replace box: 替换输入框，可在此输入替换文本。
        Replace one: 替换一个匹配项。
        Replace all: 替换所有匹配项。

    TimeDate: 插入当前日期和时间。
View:

    Zoom out: 缩小视图。
